OGC OpenSearch Geo and Time Extensions Standard
-----------------------------------------------------------------------

This OGC standard specifies the Geo and Time extensions to the 
OpenSearch query protocol. OpenSearch is a collection of simple 
formats for the sharing of search results.

More information may be found at
 http://www.opengeospatial.org/standards/opensearchgeo

The most current schema are available at http://schemas.opengis.net/
and all OGC schemas may be downloaded in a complete bundle from
http://schemas.opengis.net/SCHEMAS_OPENGIS_NET.zip
